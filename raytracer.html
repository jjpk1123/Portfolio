Ray Tracer

More info on my ray tracer!

These images are the product of a realistic recursive ray tracer rendering engine for triangulated Wavefront OBJ files or spheres. The program accepts triangulated Wavefront OBJ files, material files, and custom driver files. It performs transformation, recursive ray tracing, and refraction. Ultimately, it outputs stunning PPM P3 images. This program was written in Ross Beveridge's CS410 course at Colorado State University. He reuses the project from semester to semester, so there is no GitHub page for this project. Instead, I have a gallery! Here are a few examples!
